# PRD: Doodle Wallpaper Studio (My Pocket Atelier)

## 1. 제품 개요

**한 줄 요약:** 손그림(Neubrutalism) 스타일로 나만의 폰 배경화면을 꾸미는 모바일 앱

사용자가 다양한 위젯 타입(시간표, 원형시계, 게이지 등)을 선택하고,
캔버스 위에서 요소를 배치/꾸며서 배경화면 이미지로 저장하는 앱이다.
배경화면 기능도 있고 위젯 기능도 따로 있다.

## 2. 타겟 사용자

- 10~30대 MZ세대
- 다이어리 꾸미기(다꾸), 폰 꾸미기에 관심 있는 사용자
- 동시에 생산적인 느낌, 갓생 기능을 원하는 사용자
- 기존 배경화면 앱에서 생산적인 이미지와 세련되거나 귀여운 디자인, 자신만의 디자인을 구현하지 못했던 사람

## 3. 핵심 가치

| 가치 | 설명 |
|------|------|
| 손그림 감성 | Neubrutalism 디자인 — 굵은 테두리, 하드 섀도우, 삐뚤빼뚤 둥근 모서리 |
| 쉬운 제작 | 드래그 앤 드롭 기반, 코딩 지식 불필요 |
| 다양한 위젯과 배경화면 | 5가지 위젯 타입으로 실용성 + 꾸미기 모두 충족 |
| 생산성 | 다양한 스케줄이나 시간, 생산성 관련 내용을 시각화 |

## 4. 화면 구성

### 4-1. Home (내 작업실)
- 최근 작업물 목록 (2열 카드 그리드, 이모지 + 제목 + 날짜)
- 빈 상태일 때 격려 메시지 표시
- 하단 FAB: "+ 새로 만들기" 버튼
- 우측 상단 설정 아이콘

### 4-2. Type Selection (위젯 타입 선택)
- 상단: 배경화면 / 위젯 모드 토글
- 2열 그리드 (2-2-1 배치), 5개 타입:
  - **타임테이블** 📅: 요일별 시간표 + 원형시계 (통합)
  - **타임 게이지** ⏳: 하루의 흐름을 시각화
  - **라이프 프로그레스** 🌱: 인생의 진행률 표시
  - **모토 & 낙서** ✍️: 나만의 문구 꾸미기
  - **다꾸 믹스앤매치** 🎨: 자유롭게 다이어리 꾸미기 (풀 너비)
- 위젯 모드일 때 "인터랙티브" 뱃지 표시

### 4-3. Editor (메인 캔버스)
- **상단 툴바**: 뒤로가기, 프로젝트 제목, Undo 버튼, 저장 버튼
- **에디터 영역**: 위젯 타입별 전용 에디터 렌더링
  - 각 에디터가 자체적으로 프리뷰 + 편집 옵션을 포함

## 5. 디자인 시스템

### 색상
| 용도 | 색상 코드 |
|------|-----------|
| 배경 (기본) | `#FFFDF9` |
| 포인트 (노란) | `#FFDE59` |
| 블록 - 파랑 | `#91D0FF` / `#E6F3FF` |
| 블록 - 핑크 | `#FF91AD` / `#FFF0F5` |
| 테두리/섀도 | `#1A1A1A` |

### Neubrutalism 공통 스타일
```
border: 3px solid #1A1A1A
borderRadius: 15px 5px 12px 8px / 8px 12px 5px 15px  (삐뚤빼뚤)
boxShadow: 4px 4px 0px #1A1A1A
```

### 폰트
- **기본 (개구체)**: Google Fonts "Gaegu" (400, 700)
- **서브 (명조체)**: 추후 선정

## 6. 기술 스택

| 영역 | 기술 |
|------|------|
| Framework | React Native (Expo) |
| Styling | NativeWind (Tailwind CSS for RN) |
| Navigation | Expo Router (file-based) |
| State | Zustand |
| Icons | lucide-react-native |
| 이미지 저장 | expo-media-library + react-native-view-shot |
| 애니메이션 | react-native-reanimated |
| 제스처 | react-native-gesture-handler |
| SVG | react-native-svg |
| 저장소 | AsyncStorage (로컬 작업물 저장) |
